# Reality Cascade
Reality Cascade is an incremental journey through the evolution of existence. Start by generating basic energy, automate your growth, unlock new technologies, and transcend through layers of reality. Build powerful systems, discover new dimensions, and reshape the universe itself. This is the idea for the incremental game and will not have all the featured ideas yet, but I plan to improve and create an extraordinary game.
Play the game at https://rtcreations.github.io/reality-cascade/
